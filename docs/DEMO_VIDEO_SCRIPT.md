## üìù RECORDING TIPS

1. **Screen Recording**: Use OBS, Loom, or QuickTime
2. **Resolution**: Record at 1080p minimum
3. **Browser**: Use Chrome in fullscreen, hide bookmarks bar
4. **Zoom**: Use Ctrl+/Ctrl- to enlarge UI elements for visibility
5. **Pre-load Data**: Run `npm run seed:prod` before recording
6. **Practice**: Do 1-2 dry runs before recording
7. **Audio**: Record in a quiet room, or add voiceover in post
